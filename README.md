RandomTeleport
==============

Features
--------

This plugin will teleport a player to a random coordinate in a world that has a WorldBorder configuration.  It will attempt to avoid teleporting the player into the ocean or into lava.

Requirements
------------
- bukkit >= 1.6.0
- WorldBorder
